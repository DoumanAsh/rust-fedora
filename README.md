# rust-fedora

[![Build](https://github.com/DoumanAsh/rust-fedora/actions/workflows/docker-image.yml/badge.svg)](https://github.com/DoumanAsh/rust-fedora/actions/workflows/docker-image.yml)
[![Hub](https://img.shields.io/badge/Docker-Hub-2496ed.svg)](https://hub.docker.com/r/douman/rust-fedora/tags)

Minimal Rust container backed by fedora

## Tags

- `latest` - Stable Fedora version with latest Rust available (no rustup)
- `latest-rustup` - Stable Fedora version with latest Rust available via rustup
- `<rustc version>-fedora<fedora version>` - Specific fedora version and specific Rust available (no rustup)
- `<rustc version>-rustup-fedora<fedora version>` - Specific fedora version and specific Rust available via rustup
- `fedora<fedora version>` - Specific fedora version and latest Rust available (no rustup)
- `rustup-fedora<fedora version>` - Specific fedora version and latest Rust available via rustup
